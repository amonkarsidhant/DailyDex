# Creator — YouTube Short

Turn one research pack into a vertical short script (25–60s, ~150 spoken words max).

## Structure
1. **Cold hook (≤3s)** — a single startling number or claim. No throat-clearing.
2. **Insight** — the one thing that changed, stated plainly.
3. **Proof point** — a concrete benchmark, command, or result from the research.
4. **Close** — a punchy line that earns a follow. Imperative voice.

## Rules
- Plain spoken sentences. Read-aloud test: if it's hard to say, rewrite.
- Add `[Visual: …]` and `[Audio: …]` cues every couple of lines.
- Use real numbers and names from the research. No hype, no banned phrases.
- Output only the script. No preamble, no explanation.

This is the canonical spec mirrored by `studio.py:_prompt_shorts` and the
`shorts` skill in Creator Central.

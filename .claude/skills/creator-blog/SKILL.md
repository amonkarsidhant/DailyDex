# Creator — Blog Post

Turn one research pack into a finished, publishable Markdown post (600–900 words).

## Structure
- `# Title` — specific, no clickbait.
- Intro (2–3 sentences) stating why it matters.
- `## What it actually does` — the mechanism, concretely.
- `## Why it matters` — for the target audience.
- A fenced code block or copy-paste command when relevant.
- `## The takeaway` — the one thing to remember.

## Rules
- Full prose, not bullets-only. Technical and specific.
- Real numbers and names from the research. No hype, no banned phrases.
- Output only the post.

Canonical spec mirrored by `studio.py:_prompt_blog` and the `blog` skill.
